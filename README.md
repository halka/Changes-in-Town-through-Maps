# 地図で見る街の変化 (Changes in Town through Maps)

横須賀・三笠公園周辺を題材に、国土地理院の年代別の空中写真と現代の地図（OpenStreetMap等）を左右2画面で同期させ、街の歴史的な変化を見比べるWebアプリケーションです。

---

## 主な機能

1. **リアルタイム2画面連動**
   - 左右の地図のパン（移動）やズーム（拡大・縮小）操作が、リアルタイムに連動して同期スクロールします。

2. **豊富な地図・空中写真レイヤー**
   - **基本地図**: 標準地図（国土地理院）、淡色地図（国土地理院）、OpenStreetMap、CARTO Positron、OpenTopoMap。
   - **年代別空中写真**: 1928年（昭和3年）頃から、戦後（1945年）、高度経済成長期（1960年代〜1980年代）、最新のシームレス写真まで網羅しています。

3. **カスタム地点ピン追加 & KMLエクスポート**
   - 地図上をクリックすると地図の外にモーダルダイアログが開き、ラベル・ピン色・サイズ・文字色・フォントサイズを指定して任意の場所にピンを追加できます。
   - 追加したデータはリアルタイムで **KML** 形式として出力され、ワンクリックでダウンロードできます。
   - ピンをクリックすると詳細ポップアップから個別削除も可能です。「🗑 すべてクリア」ボタンは破渋的操作の確認モーダル付きです。

4. **プリセット・ KMLインポート**
   - プリセットインデックス（`preset/index.json`）から定義済みの地点セットを読み込んで表示できます。
   - ローカルの KML ファイルを読み込んで地図に表示することもできます。

5. **現在地にピン**
   - Geolocation API を利用して現在地の座標を取得し、両方の地図にピンをプロットできます（HTTPSまたはlocalhostなどのセキュアな環境下でのみ動作）。位置情報の取得失敗時はトースト通知で表示されます。

## 構成ファイル

- **`index.html`**
  - アプリケーションのUI構造。Leafletライブラリの読み込みが失敗した際のリトライ制御や、その他スクリプトの非同期依存関係ロードを管理しています。
- **`script.js`**
  - アプリケーションのメインロジック。Leaflet地図の初期化、双方向連動同期、ピンの動的追加・削除処理などを実装しています。
- **`style.css`**
  - レスポンシブ対応のスタイルシート。高級感のあるダーク調のコントロール類、座標表示オーバーレイ、チャット形式のイントロダクション（アコーディオン）などを定義しています。
- **`gsi-layers.js`** *(削除済み・`script.js` に統合)*
- **`export-utils.js`** *(削除済み・`script.js` に統合)*
- **`map-data.json`** *(後方互換用)*
  - 旧バージョン向けの地点データ。現バージョンでは使用されず、`preset/` 内のプリセットデータが使用されます。
- **`preset/index.json`**
  - 利用可能なプリセット一覧を定義するインデックスファイル。各エントリに `id`・`title`・`file`（プリセットJSONへの相対パス）を記述します。
- **`preset/*.json`**
  - 個別のプリセットデータファイル。`bounds`・`points` などの構造は `map-data.json` と同形式です。`title`・`description` を追加で持つことができます。

## 起動方法

### 1. Webサーバー経由で起動（推奨）
CORSエラーを回避し、`map-data.json` のロードや位置情報APIなどのフル機能を動作させるには、ローカルサーバーを介して起動することを推奨します。

- **Python 3 をお持ちの場合:**
  ターミナルで本ディレクトリへ移動し、以下のコマンドを実行します。
  ```bash
  python3 -m http.server 8000
  ```
  実行後、ブラウザで `http://localhost:8000` を開きます。

- **Node.js/npm をご利用の場合:**
  ```bash
  npx http-server
  ```

- **VS Code をご利用の場合:**
  拡張機能「Live Server」をインストールし、ステータスバーの `Go Live` ボタンをクリックしてください。

### 2. ローカルファイルを直接起動（`file://`）
`index.html` ファイルをブラウザに直接ドラッグ＆ドロップして開くことも可能です。
*※この場合、CORS制限によりプリセットデータの読み込みが失敗するため初期ピンは表示されません。また「現在地にピン」機能もセキュアコンテキストが必要なため動作しません。カスタムピンの追加・KMLファイルの読み込みは引き続き利用できます。*
